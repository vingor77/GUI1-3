Input restrictions:

  • Letters are not accepted.
  • Integers greater than 50 or less than -50 are not accepted.
    • If an integer is greater than 50, it will be set to 50.
    • If an integer is less than -50, it will be set to -50.
  • Decimal numbers will be rounded to the nearest whole number.
  • The minimum column value must be less than the maximum column value.
  • The minimum row value must be less than the maximum row value.
